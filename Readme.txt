Пользователь работает с файлами в своей папке и перидически ему нужно сбрасывать их по путям, которые он в душе не чает.
ИТэшники настраивают ему инишник, где расписаны названия задач и откуда - куда скидывать.
Скрипт проверяет, есть ли файл, сранвивает время последнего изменения и копирует.

Для того чтобы считать ini файл он должен лежать в каталоге программы, так как путь к нему берется из пути к запускаемому файлу.
